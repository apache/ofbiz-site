Release Notes 24.09.05

home / Download /Release Notes 24.09.05

Apache OFBiz® 24.09.05, released in January 2026, is the fifth release of the 24.09 series, which has been feature-frozen since September 2024, receiving only bug fixes.

Release Notes - OFBiz - Version 24.09.05

Bug

  • [OFBIZ-13070] - BalanceSheet generates an error
  • [OFBIZ-13283] - Canceling a partially received Purchase Order receipt throws a "Required Field Missing: Receipt Id" error.
  • [OFBIZ-13317] - Tried to execute unauthorized script
  • [OFBIZ-13320] - In several locations strings are compared with == and !=
  • [OFBIZ-13321] - getClass() is called on instance of Class
  • [OFBIZ-13323] - service copyInvoice Failed on invoice not in process
  • [OFBIZ-13325] - Invalid characters encountered
  • [OFBIZ-13327] - Fix various bugs in receive against purchase orders
  • [OFBIZ-13330] - Compound's single form are loaded as ModelGrid in memory
  • [OFBIZ-13332] - Lookup don't work on area just updated
  • [OFBIZ-13333] - Bad request parameter encoding with tomcat 9
  • [OFBIZ-13335] - Purchase Order approval fails when payment auto create config is disabled
  • [OFBIZ-13337] - 'New Shipment For Ship Group' buttons should be available only after POs have been approved
  • [OFBIZ-13339] - jsgantt-improved bloks qs.js update

Improvement

  • [OFBIZ-13257] - The "Select" button in the Calendar Exception Day section functions as an edit action but lacks appropriate context and UI feedback, leading to user confusion
