Release Notes 17.12.07

home/Download/Release Notes 17.12.07

Apache OFBiz® 17.12.07, released on April 2021, is the seventh and final release of the 17.12 series, that has been stabilized since December 2017.

Sub-task

  • [OFBIZ-12080] - Secure the uploads
  • [OFBIZ-12205] - Upgrade Apache PDFBox to 2.0.23 because of CVE-2021-27807 and CVE-2021-27906
  • [OFBIZ-12212] - Comment out the SOAP and HTTP engines

Bug

  • [OFBIZ-12175] - Party Export/Import not congruent
  • [OFBIZ-12211] - addImageForProduct fails
  • [OFBIZ-12222] - Error while deleting a Facility because of an existing FacilityLocationGeoPoint

Improvement

  • [OFBIZ-12216] - Improve UtilObject class
  • [OFBIZ-12221] - Improve ObjectInputStream denyList
