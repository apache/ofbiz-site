Apache OFBiz is a suite of business applications flexible enough to be used across any industry. A common architecture allows developers to easily extend or enhance it to create custom features.

Java

Powerful Java Web Framework

OFBiz is a Java based web framework including an entity engine, a service engine and a widget based UI allowing you to quickly prototype and develop your web application.

CRM & ERP

Mature CRM & ERP Solution

An Apache top level project for 10 years, OFBiz has shown it's stability and maturity as an enterprise-wide ERP solution that is flexible enough to change with your business.

Develop

Developer Friendly

OFBiz architecture is extremely flexible allowing developers to quickly and easily extend and enhance the framework with custom features.

Where to Start?

Developer or Business User, getting started with OFBiz is easy!. We have a range of information about OFBiz modules and features as well as an online demo that you can try.

  • Business Users can get started on our Business Users page

Features and Benefits

Modules

Apache OFBiz comes with a range of core modules out-of-the-box (OOTB) including:

  1. Accounting (GL,AR,AP,FA)
  2. CRM,Order Management & E-Commerce
  3. Warehousing and Inventory
  4. Manufacturing & MRP

Features

Apache OFBiz has a full set useful business features including:

  1. Product & Catalog Management
  2. Promotion & Pricing Management
  3. Supply Chain Fulfillment
  4. Contracts, Payments & Billing

Benefits

There are a range of benefits associated with using Apache OFBiz including:.

  1. Free and open source - No upfront costs
  2. Scalable, reliable enterprise solution
  3. Fully Customisable
  4. Flexible to grow with your business

If you are still unsure or have any questions then please feel free to use our User Mailing List
